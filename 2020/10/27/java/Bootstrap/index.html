Bootstrap

概念：前端开发框架，Bootstrap，来自Twitter，是目前很受欢迎的前端框架。Bootstrap是基于HTML、CSS、JavaScript的

  • 框架：半成品软件，开发人员可以在框架基础上，进行开发，简化编程
  • 优点：
    • 定义很多CSS样式和JS插件。可以直接CV
    • 响应式布局：同一套页面兼容不同分辨率的设备

响应式布局

  • 同一页面兼容不同分辨率的设备
  • 实现∶依赖于栅格系统:将一行平均分成12个格子，可以指定元素占几个格子
  • 步骤︰
    1. 定义容器，相当于之前的table
      • 容器分类：
        • container：两边留白
        • container-fluid：每一种设备都是100%宽度
    2. 定义行。相当于之前的tr 样式：row
    3. 定义元素。指定该元素在不同的设备上，所占的格子数目。样式:col-设备代号-格子数目
      • 设备代号：
        • xs：超小屏幕手机(<768px) : col-xs-122.
        • sm :小屏幕平板(>768px)
        • md :中等屏幕桌面显示器(2992px)
        • lg:大屏幕大桌面显示器(21200px)
  • 注意：
    • 如果一行中格子数量超过12，超出部分自动换行
    • 栅格类属性可以向上兼容
    • 如果真实设备宽度小于了设置棚格类属性的设备代码的最小值，会一个元素沾满一整行。
