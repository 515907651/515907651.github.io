HTML

Hyper Text Markup Language 超文本标记语言

超文本：超文本是用超链接的方法，将各种不同空间的文字信息组织在一起的网状文本

标记语言：由标签构成的语言，不是编程语言

基本语法

  1. html文档后缀名.html或者.htm

  2. 标签分为

    1. 围堵标签：有开始和结束标签。
    2. 自闭合标签：开始标签和结束标签在一起。

  3. 标签可以嵌套

    错误嵌套：

    正确嵌套：

  4. 在开始标签中可以定义属性。属性由键值对构成，值需要引号（单双都可）引起来

  5. html的标签不分大小写，建议小写

标签学习

  1. 文件标签：构成html最基本的根标签

    • html：html文件的根标签
    • head：头标签。用于指定html文件的一些属性，引入外部资源
    • title：标题标签
    • body：体标签
    • !DOCTYPE：html5中定义该文档是html文档，两边加<>

  2. 文本标签：和文本有关的标签

    • 注释：

    • h1 to h6：标题标签，字体依次变小

    • p：换行标签

    • hr：展示一条水平线

      属性：

      • color
      • width
      • size
      • align：center， left，right

    • b：字体加粗

    • i：字体斜体

    • font：字体标签

    • center：文本居中

    • 属性定义：

      • color：
        • 英文单词：red，green等
        • rgb（值1，值2，值3）：值的范围：0-255
        • #值1值2值3：值的范围：00-FF
      • width：
        • 数值：单位是像素
        • 数值%：占比相对于父元素的比例

  3. 图片标签：

    ima source=路径

    路径：

    • 绝对路径
    • 相对路径：
      • ./ 代表当前目录
      • ../ 代表上一级目录

  4. 列表标签：

    • 有序列表：ol，li代表行
    • 无序列表：ul，li代表行

  5. 连接标签：

    a href=”指定资源”URL

    target：

    • _self：在当前页面跳转
    • _blank：空页面打开

  6. div 和 span

    结合css使用

    span中的内容没有任何效果，只能起一个包含的作用，文字不换行，块级标签

    div自动换行，内联标签

  7. 语义化标签：为了提高可读性

    • header
    • footer

  8. 表格标签：

    • table：定义表格

      • width：宽度
      • border：边框
      • cellpadding：定义内容和单元格之间的距离
      • cellspacing：定义单元格之间的距离。如果指定为0，单元格的线为0
      • bgcolor：背景色
      • align：对齐方式

    • tr：行

      • bgcolor：背景颜色
      • align：对其方式

    • td：单元格

      • colspan：合并列
      • rowspan：合并行

    • th：表头单元格

    • caption：表格标题

    • tread：表格的头部分

    • tbody：表格的体部分

    • tfoot：表格的脚

  9. 表单标签

    • 概念：用于采集用户数据，和服务器进行交互
    • form：用于定义表单的。可以定义一个范围，范围代表采集用户数据的范围
    • 属性：
      • action：指定提交数据的URL
      • method：指定提交方式
        • get：
          • 请求参数会在地址栏显示。封装在请求中
          • 请求参数大小有限制
          • 不安全
        • post：
          • 请求参数不在地址栏显示。封装在请求体中
          • 参数大小没有限制
          • 安全
    • 表单项中的数据要想被提交：必须指定name参数
  • input：可以通过type属性值，改变元素展示的样式
    • type属性：
      • text：文本输入框，默认值
        • placeholder：指定输入框的提示信息，当输入框的内容发生变化时，会自动清空提示信息
      • password：密码输入框
      • radio：单选框
        • 想要让多个单选框实现单选的效果，则多个单选框的name属性必须一致
        • 指定value属性，指定其被选中后提交的值
        • checked属性，指定默认值
      • checkbox：复选框
        • 一般会给每一个单选框提供value属性，指定其被选中后提交的值
        • checked属性，指定默认值
    • label：指定输入项的文字描述信息
      • label属性一般会和input的id属性对应。如果对应了，点击label区域，会让input输入框获取焦点
  • select：下拉列表
    • 子元素：option，指定列表项
  • textarea：文本域
    • cols：指定列数，每一行有多少字符
    • rows：默认多少行
